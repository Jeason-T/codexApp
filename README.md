# 番茄章节批量上传助手

本项目是一个本地 Windows 桌面工具，目标是辅助作者整理并上传本人作品章节到番茄作家后台。第一步只完成 Electron + Vue3 基础项目和界面骨架。

## 当前进度

- Electron 主窗口
- Vue3 基础界面
- 顶部状态栏
- 文件拖拽区占位
- 章节列表占位
- 正文预览占位
- 底部操作区和日志面板占位

## 运行

```bash
npm install
npm run dev
```

## 合规边界

- 不保存账号密码
- 不绕过验证码或人机验证
- 不逆向平台私有接口
- 默认只保存草稿
- 遇到登录或验证必须由用户手动处理
